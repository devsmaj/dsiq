# DSIQ Architecture

## Repository Layout

- `frontend`: Next.js web app
- `backend`: API and business logic
- `docs`: project documentation
- `README.md`: project overview
- `TODO.md`: implementation checklist

## Frontend Layout

- `src/app`: routes, layouts, and page entrypoints
- `src/components`: shared UI building blocks
- `src/features`: feature-owned modules
- `src/lib`: helpers, clients, config, and utilities

## Backend Layout

- `src/config`: runtime configuration
- `src/routes`: top-level API routing
- `src/modules`: domain modules with controller/service/routes separation
